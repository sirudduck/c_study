# Session 08 - 구조체 복습 문제

---

## 문제

아래 조건에 맞는 프로그램을 작성하라.

---

### ① 구조체 정의

`typedef`를 사용하여 아래 멤버를 가진 `Movie` 구조체를 정의하라.

| 멤버 이름 | 타입       | 설명               |
|-----------|------------|--------------------|
| `title`   | `char[30]` | 영화 제목          |
| `year`    | `int`      | 개봉 연도          |
| `rating`  | `float`    | 평점 (0.0 ~ 10.0)  |

---

### ② 구조체 배열 초기화

`Movie` 구조체 배열 3개를 선언하고 아래 데이터로 초기화하라.

| 제목         | 개봉 연도 | 평점 |
|--------------|-----------|------|
| Interstellar | 2014      | 8.7  |
| Inception    | 2010      | 8.8  |
| Tenet        | 2020      | 7.4  |

---

### ③ 함수 작성

아래 두 함수를 작성하라.

```c
// 구조체 배열 전체를 출력하는 함수
void print_all(Movie *arr, int n);

// 평점이 가장 높은 영화의 포인터를 반환하는 함수
Movie *find_best(Movie *arr, int n);
```

---

### ④ main 작성

- `print_all`로 전체 목록을 출력한다.
- `find_best`의 반환값을 받아 가장 평점 높은 영화의 제목과 평점을 출력한다.

**예상 출력:**
```
[2014] Interstellar - 8.7점
[2010] Inception - 8.8점
[2020] Tenet - 7.4점

최고 평점: Inception (8.8점)
```

---

## 점검 포인트

- `typedef struct` 문법이 올바른가
- `.` 와 `->` 연산자를 상황에 맞게 구분하여 사용했는가
- `find_best`가 포인터를 반환하고, `main`에서 `->` 로 멤버에 접근하는가
- `print_all`에 배열 이름(포인터)과 크기를 올바르게 전달했는가
