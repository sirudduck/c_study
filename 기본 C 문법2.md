
## 3. 관계 연산자 (Relational Operators)

조건을 표현할 때 사용하는 연산자입니다.

| 연산자 | 의미 | 예제 | 결과 (a=5, b=3일 때) |
|--------|------|------|---------------------|
| `==` | 같다 | `a == b` | 거짓 (0) |
| `!=` | 같지 않다 | `a != b` | 참 (1) |
| `>` | 크다 | `a > b` | 참 (1) |
| `<` | 작다 | `a < b` | 거짓 (0) |
| `>=` | 크거나 같다 | `a >= 5` | 참 (1) |
| `<=` | 작거나 같다 | `b <= 3` | 참 (1) |

**중요**: 
- 같은지 비교: `==` (등호 2개)
- 대입: `=` (등호 1개)

```c
// ❌ 잘못된 예
if (age = 19)  // 대입 (age에 19를 저장)

// ✅ 올바른 예
if (age == 19)  // 비교 (age가 19와 같은지)
```

### else if 문

**형식**
```c
if (조건1) {
    // 조건1이 참일 때
} else if (조건2) {
    // 조건1이 거짓이고, 조건2가 참일 때

    // 조건1, 조건2가 거짓이고, 조건3이 참일 때

    // 모든 조건이 거짓일 때
}
```

**실행 흐름**
1. 위에서부터 순서대로 조건 검사
2. 참인 조건을 만나면 해당 블록 실행
3. 나머지는 건너뜀 (하나만 실행됨)

## 5. 논리 연산자 (Logical Operators)

여러 조건을 조합할 때 사용합니다.

| 연산자 | 의미 | 예제 | 의미 |
|--------|------|------|------|
| `&&` | AND (그리고) | `a > 0 && a < 10` | a가 0보다 크고 10보다 작다 |
| `||` | OR (또는) | `a < 0 || a > 100` | a가 0보다 작거나 100보다 크다 |
| `!` | NOT (부정) | `!(a == 0)` | a가 0이 아니다 |

### 진리표

#### AND (`&&`) - 둘 다 참이어야 참

| 조건1 | 조건2 | 결과 |
|-------|-------|------|
| 참 | 참 | **참** |
| 참 | 거짓 | 거짓 |
| 거짓 | 참 | 거짓 |
| 거짓 | 거짓 | 거짓 |

#### OR (`||`) - 하나라도 참이면 참

| 조건1 | 조건2 | 결과 |
|-------|-------|------|
| 참 | 참 | **참** |
| 참 | 거짓 | **참** |
| 거짓 | 참 | **참** |
| 거짓 | 거짓 | 거짓 |

### 예제: 범위 검사

```c
// 0 이상 100 이하인지 검사
int score = 85;

if (score >= 0 && score <= 100) {
    printf("유효한 점수입니다.\n");
} else {
    printf("잘못된 점수입니다.\n");
}
```

```c
// 주말인지 검사 (토요일=6, 일요일=0)
int day = 6;

if (day == 0 || day == 6) {
    printf("주말입니다!\n");
} else {
    printf("평일입니다.\n");
}
```

## 7. switch-case 문

### 언제 사용하나?

- 하나의 변수가 **여러 값 중 하나**와 일치하는지 검사할 때
- `if-else if` 체인이 길어질 때

### 기본 형식

```c
switch (변수) {
    case 값1:
        // 변수 == 값1일 때 실행
        break;
    case 값2:
        // 변수 == 값2일 때 실행
        break;
    case 값3:
        // 변수 == 값3일 때 실행
        break;
    default:
        // 어느 case에도 해당하지 않을 때 실행
}
```

**중요**: `break`를 빠뜨리면 다음 case도 실행됩니다! (fall-through)

### 예제: 메뉴 선택

```c
#include <stdio.h>

int main(void) {
    int choice;
    
    printf("=== 메뉴 ===\n");
    printf("1. 아메리카노\n");
    printf("2. 카페라떼\n");
    printf("3. 에스프레소\n");
    printf("선택: ");
    scanf("%d", &choice);
    
    switch (choice) {
        case 1:
            printf("아메리카노를 선택하셨습니다.\n");
            break;
        case 2:
            printf("카페라떼를 선택하셨습니다.\n");
            break;
        case 3:
            printf("에스프레소를 선택하셨습니다.\n");
            break;
        default:
            printf("잘못된 선택입니다.\n");
    }
    
    return 0;
}
```

## 8. 계산기 업그레이드 (문제 1-4 해답)

```c
#include <stdio.h>

int main(void) {
    double num1, num2, result;
    char op;
    
    printf("첫 번째 숫자: ");
    scanf("%lf", &num1);
    
    printf("두 번째 숫자: ");
    scanf("%lf", &num2);
    
    printf("연산자 (+, -, *, /): ");
    scanf(" %c", &op);  // 앞에 공백 주의!
    
    switch (op) {
        case '+':
            result = num1 + num2;
            printf("%.2f + %.2f = %.2f\n", num1, num2, result);
            break;
        case '-':
            result = num1 - num2;
            printf("%.2f - %.2f = %.2f\n", num1, num2, result);
            break;
        case '*':
            result = num1 * num2;
            printf("%.2f * %.2f = %.2f\n", num1, num2, result);
            break;
        case '/':
            if (num2 == 0) {
                printf("0으로 나눌 수 없습니다.\n");
            } else {
                result = num1 / num2;
                printf("%.2f / %.2f = %.2f\n", num1, num2, result);
            }
            break;
        default:
            printf("잘못된 연산자입니다.\n");
    }
    
    return 0;
}
```

**실행 예시 1**
```
첫 번째 숫자: 10
두 번째 숫자: 3
연산자 (+, -, *, /): /

```

**실행 예시 2 (0으로 나누기)**
```
첫 번째 숫자: 5
두 번째 숫자: 0
```

# Part 4: 반복문 배우기

## 1. for 문 (정해진 횟수만큼 반복)

### 기본 형식

```c
for (초기화; 조건; 증감) {
    // 반복할 코드
}
```

**실행 순서**
1. 초기화 (처음 1번만)
2. 조건 검사 (참이면 3번으로, 거짓이면 종료)
3. 반복할 코드 실행
4. 증감
5. 2번으로 돌아감

### 예제: 1부터 5까지 출력

```c
#include <stdio.h>

int main(void) {
    int i;
    
    for (i = 1; i <= 5; i++) {
        printf("%d\n", i);
    }
    
    return 0;
}
```

**실행 흐름**
```
i = 1 시작
i <= 5? 참 → 1 출력 → i++ (i = 2)
i <= 5? 참 → 2 출력 → i++ (i = 3)
i <= 5? 참 → 3 출력 → i++ (i = 4)
i <= 5? 참 → 4 출력 → i++ (i = 5)
i <= 5? 참 → 5 출력 → i++ (i = 6)
i <= 5? 거짓 → 종료
```

**출력**
```
1
2
3
4
5
```

## 3. 1부터 N까지의 합 (문제 2-3 해답)

```c
#include <stdio.h>

int main(void) {
    int n, i, sum = 0;
    
    printf("N을 입력하세요: ");
    scanf("%d", &n);
    
    for (i = 1; i <= n; i++) {
        sum += i;  // sum = sum + i
    }
    
    printf("1부터 %d까지의 합: %d\n", n, sum);
    
    return 0;
}
```

**실행 흐름 (n=5일 때)**
```
sum = 0 초기화
i = 1: sum = 0 + 1 = 1
i = 2: sum = 1 + 2 = 3
i = 3: sum = 3 + 3 = 6
i = 4: sum = 6 + 4 = 10
i = 5: sum = 10 + 5 = 15
결과: 15
```

## 5. 중첩 for 문 (Nested Loop)

for 문 안에 또 다른 for 문을 넣을 수 있습니다.

### 예제: 전체 구구단 (2단~9단)

```c
#include <stdio.h>

int main(void) {
    int dan, i;
    
    for (dan = 2; dan <= 9; dan++) {
        printf("\n=== %d단 ===\n", dan);
        for (i = 1; i <= 9; i++) {
            printf("%d x %d = %d\n", dan, i, dan * i);
        }
    }
    
    return 0;
}
```

**실행 흐름**
```
dan = 2:
  i = 1: 2 x 1 = 2
  i = 2: 2 x 2 = 4
  ...
  i = 9: 2 x 9 = 18
dan = 3:
  i = 1: 3 x 1 = 3
  ...
```

## 6. while 문 (조건이 참인 동안 반복)

### 기본 형식

```c
while (조건) {
    // 조건이 참인 동안 반복
}
```

**for vs while**
- `for`: 반복 횟수를 **알 때**
- `while`: 반복 횟수를 **모를 때**, 특정 조건까지 반복

### 예제: 1부터 5까지 출력 (while 버전)

```c
#include <stdio.h>

int main(void) {
    int i = 1;  // 초기화
    
    while (i <= 5) {  // 조건
        printf("%d\n", i);
        i++;  // 증감 (직접 작성!)
    }
    
    return 0;
}
```

**주의**: `i++`를 빼먹으면 **무한 루프**!

## 7. while 문 활용 예제

### 예제 1: 입력받은 값이 0이 될 때까지 합 계산

```c
#include <stdio.h>

int main(void) {
    int num, sum = 0;
    
    printf("숫자를 입력하세요 (0을 입력하면 종료):\n");
    
    while (1) {  // 무한 루프
        scanf("%d", &num);
        
        if (num == 0) {
            break;  // 반복문 탈출
        }
        
        sum += num;
        printf("현재 합계: %d\n", sum);
    }
    
    printf("최종 합계: %d\n", sum);
    
    return 0;
}
```

**실행 예시**
```
숫자를 입력하세요 (0을 입력하면 종료):
5
현재 합계: 5
10
현재 합계: 15
-3
현재 합계: 12
0
최종 합계: 12
```

### break와 continue

| 키워드 | 기능 |
|--------|------|
| `break` | 반복문을 **즉시 종료** |
| `continue` | 현재 반복을 **건너뛰고** 다음 반복으로 |

### 예제 2: continue 사용

```c
// 1부터 10까지 중 짝수만 출력
#include <stdio.h>

int main(void) {
    int i;
    
    for (i = 1; i <= 10; i++) {
        if (i % 2 != 0) {  // 홀수면
            continue;       // 건너뛰기
        }
        printf("%d\n", i);
    }
    
    return 0;
}
```

**출력**
```
2
4
6
8
10
```

## 8. do-while 문 (최소 1번은 실행)

### 기본 형식

```c
do {
    // 최소 1번은 실행
} while (조건);
```

**차이점**
- `while`: 조건을 **먼저** 검사 → 0번 실행 가능
- `do-while`: 코드를 **먼저** 실행 → 최소 1번 실행

### 예제: 메뉴 프로그램

```c
#include <stdio.h>

int main(void) {
    int choice;
    
    do {
        printf("\n=== 메뉴 ===\n");
        printf("1. 더하기\n");
        printf("2. 빼기\n");
        printf("3. 곱하기\n");
        printf("4. 나누기\n");
        printf("0. 종료\n");
        printf("선택: ");
        scanf("%d", &choice);
        
        if (choice >= 1 && choice <= 4) {
            printf("계산 기능은 나중에 구현할 예정입니다.\n");
        } else if (choice == 0) {
            printf("프로그램을 종료합니다.\n");
        } else {
            printf("잘못된 선택입니다.\n");
        }
        
    } while (choice != 0);
    
    return 0;
}
```

### 문제 3-5: 숫자 맞추기 게임

**문제**  
1부터 100 사이의 임의의 숫자를 정하고, 사용자가 맞출 때까지 힌트를 주는 프로그램을 작성하세요.

**실행 예시**
```
1부터 100 사이의 숫자를 맞춰보세요!
예상: 50
더 큰 숫자입니다.
예상: 75
더 작은 숫자입니다.
예상: 63
정답입니다! 3번 만에 맞추셨습니다.
```

**구현 방법 (간단 버전)**
```c
#include <stdio.h>

int main(void) {
    int answer = 63;  // 정답 (임의로 설정)
    int guess, count = 0;
    
    printf("1부터 100 사이의 숫자를 맞춰보세요!\n");
    
    while (1) {
        printf("예상: ");
        scanf("%d", &guess);
        count++;
        
        if (guess < answer) {
            printf("더 큰 숫자입니다.\n");
        } else if (guess > answer) {
            printf("더 작은 숫자입니다.\n");
        } else {
            printf("정답입니다! %d번 만에 맞추셨습니다.\n", count);
            break;
        }
    }
    
    return 0;
}
```

**참고**: 진짜 랜덤 숫자는 나중에 `rand()` 함수를 배우면 만들 수 있습니다.

## 📌 조건문 vs 반복문 정리

### 조건문 (Conditional)

| 종류 | 언제 사용? | 형식 |
|------|-----------|------|
| `if` | 단일 조건 | `if (조건) { }` |
| `if-else` | 양자택일 | `if (조건) { } else { }` |
| `if-else if` | 다중 조건 | `if (조건1) { } else if (조건2) { }` |
| `switch-case` | 정확히 일치하는 값 비교 | `switch (변수) { case 값: }` |

### 반복문 (Loop)

| 종류 | 언제 사용? | 형식 |
|------|-----------|------|
| `for` | 반복 횟수를 알 때 | `for (초기화; 조건; 증감) { }` |
| `while` | 조건 기반 반복 | `while (조건) { }` |
| `do-while` | 최소 1번 실행 | `do { } while (조건);` |

### 제어문

| 키워드 | 기능 | 사용 위치 |
|--------|------|----------|
| `break` | 반복문 즉시 탈출 | `for`, `while`, `do-while`, `switch` |
| `continue` | 현재 반복 건너뛰기 | `for`, `while`, `do-while` |

## 🎓 요약

### 오늘 배운 것

1. **조건문**
   - `if`, `else`, `else if`
   - 관계 연산자: `==`, `!=`, `>`, `<`, `>=`, `<=`
   - 논리 연산자: `&&`, `||`, `!`
   - `switch-case`

2. **반복문**
   - `for` 문: 정해진 횟수
   - `while` 문: 조건 기반
   - `do-while` 문: 최소 1번 실행
   - 중첩 루프

3. **제어문**
   - `break`: 탈출
   - `continue`: 건너뛰기

### 실전 패턴

```c
// 패턴 1: 범위 검사
if (score >= 0 && score <= 100) { }

// 패턴 2: 값 비교
switch (choice) {
    case 1: break;
    default:
}

// 패턴 3: 정해진 횟수 반복
for (i = 0; i < n; i++) { }

// 패턴 4: 조건 기반 반복
while (input != 0) { }

// 패턴 5: 최소 1번 실행
do { } while (choice != 0);
```
