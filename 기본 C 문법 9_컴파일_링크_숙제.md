# Session 09 숙제 - 오류를 찾아서 고쳐라

> 이 숙제는 **일부러 오류가 있는 코드**를 제공한다.  
> 빌드하여 오류 메시지를 확인하고, 원인을 파악한 뒤 수정하는 것이 목표다.

---

## 사전 준비

아래 파일들을 Visual Studio 프로젝트에 **모두** 추가한다.  
각 문제는 `#if 0` / `#endif`로 감싸져 있으므로, 풀고 싶은 문제의 `#if 0`을 `#if 1`로 바꿔 활성화한다.

> **주의:** 한 번에 하나의 문제만 활성화할 것. 여러 문제를 동시에 켜면 `main` 함수가 충돌한다.

---

## 공통 파일: calc.h

이 파일은 모든 문제에서 공통으로 사용한다.  
**이 파일은 수정하지 않는다** (문제에서 별도로 지시하는 경우 제외).

```c
/* calc.h */
#ifndef CALC_H
#define CALC_H

typedef struct {
    char name[30];
    int  math;
    int  english;
    int  science;
} Student;

float calc_total(Student *s);
float calc_average(Student *s);
void  print_student(Student *s);

#endif
```

## 공통 파일: calc.c

```c
/* calc.c */
#include <stdio.h>
#include "calc.h"

float calc_total(Student *s) {
    return (float)(s->math + s->english + s->science);
}

float calc_average(Student *s) {
    return calc_total(s) / 3.0f;
}

void print_student(Student *s) {
    printf("이름: %s\n", s->name);
    printf("수학: %d, 영어: %d, 과학: %d\n",
           s->math, s->english, s->science);
    printf("총점: %.0f, 평균: %.1f\n",
           calc_total(s), calc_average(s));
}
```

## 제출 방법

각 문제에 대해 아래 내용을 작성하여 제출한다.

**① 오류 메시지 기록** — 빌드 시 나타난 오류 메시지를 그대로 복사  
**② 오류 원인 분석** — 왜 이 오류가 발생했는지 한 줄로 설명  
**③ 수정 내용** — 어떤 파일의 어떤 부분을 어떻게 고쳤는지 기록  
**④ 수정 후 실행 결과** — 정상 빌드 후 출력 결과를 복사
