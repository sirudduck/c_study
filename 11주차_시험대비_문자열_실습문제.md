### C-2 ⭐⭐ — 문자열 뒤집기 (in-place)
주어진 문자열을 그 자리에서 뒤집는 `reverse_string(char *s)`를 작성하시오.
- 예: `"hello"` → `"olleh"`

<details>
<summary>👉 정답 예시</summary>

```c
#include <stdio.h>
#include <string.h>

void reverse_string(char *s) {
    int n = strlen(s);
    for (int i = 0; i < n / 2; i++) {
        char tmp   = s[i];
        s[i]       = s[n - 1 - i];
        s[n-1-i]   = tmp;
    }
}

int main(void) {
    char s[] = "Hello, World!";
    reverse_string(s);
    printf("%s\n", s);                  // !dlroW ,olleH
    return 0;
}
```

**시험 포인트:** `n/2`까지만 도는 것이 핵심. 끝까지 돌면 다시 원위치.

</details>

### C-3 ⭐⭐ — 단어 개수 세기
공백·구두점으로 구분된 문자열에서 단어 개수를 세는 `count_word(const char *s)`를 작성하시오.
- 예: `"the c book..."` → `3`

<details>
<summary>👉 정답 예시</summary>

```c
#include <stdio.h>
#include <ctype.h>

int count_word(const char *s) {
    int wc = 0, in_word = 0;
    for (int i = 0; s[i] != '\0'; i++) {
        if (isalpha((unsigned char)s[i])) {
            if (!in_word) { wc++; in_word = 1; }
        } else {
            in_word = 0;
        }
    }
    return wc;
}

int main(void) {
    printf("%d\n", count_word("the c book..."));   // 3
    printf("%d\n", count_word("hello  world!!"));  // 2
    return 0;
}
```

**시험 포인트:** "공백 다음 알파벳에서만 카운트". 상태 변수 `in_word` 패턴을 기억하자.

</details>

### C-4 ⭐⭐ — 비밀번호 강도 검증
다음 조건을 모두 만족하면 "강한 암호", 아니면 "유효하지 않은 암호"를 출력하는 프로그램을 작성하시오.
- 길이 7자 이상
- 소문자 1개 이상
- 대문자 1개 이상
- 숫자 1개 이상

<details>
<summary>👉 정답 예시</summary>

```c
#include <stdio.h>
#include <string.h>
#include <ctype.h>

int main(void) {
    char pass[100];
    printf("패스워드: ");
    gets_s(pass, sizeof(pass));

    int lower = 0, upper = 0, digit = 0;
    for (int i = 0; pass[i] != '\0'; i++) {
        if (islower((unsigned char)pass[i])) lower++;
        if (isupper((unsigned char)pass[i])) upper++;
        if (isdigit((unsigned char)pass[i])) digit++;
```

### C-5 ⭐⭐ — 회문(Palindrome) 검사
앞뒤가 같은 문자열인지 검사하는 `is_palindrome(const char *s)`를 작성하시오. 대소문자 무시.
- 예: `"Level"` → 1, `"Hello"` → 0

<details>
<summary>👉 정답 예시</summary>

```c
#include <stdio.h>
#include <string.h>
#include <ctype.h>

int is_palindrome(const char *s) {
    int left = 0, right = strlen(s) - 1;
    while (left < right) {
        if (tolower((unsigned char)s[left]) !=
            tolower((unsigned char)s[right])) {
            return 0;
        }
        left++;
        right--;
    }
    return 1;
}

int main(void) {
    printf("%d\n", is_palindrome("Level"));   // 1
    printf("%d\n", is_palindrome("Hello"));   // 0
    printf("%d\n", is_palindrome("Madam"));   // 1
    return 0;
}
```

**시험 포인트:** 양 끝에서 안쪽으로 좁히는 **투 포인터** 패턴(10주차 배열 뒤집기와 동일 구조).

</details>

### C-6 ⭐⭐⭐ — 시저 암호화·복호화
영문 알파벳을 `shift`만큼 시프트하는 `encrypt`와 그 역연산 `decrypt`를 작성하시오. 대소문자 모두 처리하고, `shift`가 26 이상이거나 음수여도 정상 동작해야 한다.

<details>
<summary>👉 정답 예시</summary>

```c
#include <stdio.h>

void caesar(char *s, int shift) {
    shift = ((shift % 26) + 26) % 26;     // 음수·26 이상 보정
    for (int i = 0; s[i] != '\0'; i++) {
        if (s[i] >= 'a' && s[i] <= 'z')
            s[i] = (s[i] - 'a' + shift) % 26 + 'a';
        else if (s[i] >= 'A' && s[i] <= 'Z')
            s[i] = (s[i] - 'A' + shift) % 26 + 'A';
    }
}

void encrypt(char *s, int shift) { caesar(s, +shift); }
void decrypt(char *s, int shift) { caesar(s, -shift); }

int main(void) {
    char msg[] = "Meet At Midnight";
    encrypt(msg, 3);
    printf("암호화: %s\n", msg);          // Phhw Dw Plgqljkw
    decrypt(msg, 3);
    printf("복호화: %s\n", msg);          // Meet At Midnight
    return 0;
}
```

**시험 포인트:** `% 26`으로 wrap-around. PDF 코드의 단순 `-= 26` 방식은 `shift`가 클 때 깨진다.

</details>

### C-10 ⭐⭐⭐ — Mini Project: 행맨(Hangman)
미리 정의된 단어 5개 중 하나를 무작위로 골라, 사용자가 알파벳을 한 글자씩 추측하게 한다.
- 시도 횟수는 최대 8회.
- 정답을 모두 맞히면 "성공", 횟수 초과 시 "실패".

<details>
<summary>👉 정답 예시</summary>

```c
#include <stdio.h>
#include <string.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    const char *words[] = {
        "programming", "language", "computer", "keyboard", "memory"
    };
    srand((unsigned)time(NULL));
    const char *answer = words[rand() % 5];
    int len = strlen(answer);

    char display[64];
    for (int i = 0; i < len; i++) display[i] = '_';
    display[len] = '\0';

    int tries_left = 8;
    while (tries_left > 0) {
        printf("\n현재: %s   (남은 기회: %d)\n", display, tries_left);
        printf("글자 추측: ");
        char ch;
        scanf(" %c", &ch);

        int hit = 0;
        for (int i = 0; i < len; i++) {
            if (answer[i] == ch && display[i] == '_') {
                display[i] = ch;
                hit = 1;
            }
        }
        if (!hit) {
            tries_left--;
            printf("틀렸습니다!\n");
        }
        if (strcmp(display, answer) == 0) {
            printf("성공! 정답: %s\n", answer);
            return 0;
        }
    }
    printf("실패. 정답은 %s 였습니다.\n", answer);
    return 0;
}
```

**시험 포인트:**
- 정답과 표시 문자열을 **두 개의 배열**로 분리 관리.
- 일치 검사는 `strcmp(display, answer) == 0`.
- `scanf(" %c", ...)`의 앞 공백 — 이전 입력의 줄바꿈 문자를 건너뛰기 위함.

**도전:** ① 같은 글자 중복 입력 차단 ② 대소문자 무시 ③ 단어 카테고리 추가

</details>

### D-1 ⭐⭐⭐ — `my_strstr` 직접 구현
`s1` 안에서 `s2`가 처음 나타나는 위치의 **포인터**를 반환하는 함수를 작성하시오. 없으면 `NULL`.

```c
char *my_strstr(const char *s1, const char *s2);
```

<details>
<summary>👉 정답 예시</summary>

```c
#include <stdio.h>

char *my_strstr(const char *s1, const char *s2) {
    if (*s2 == '\0') return (char *)s1;

    for (int i = 0; s1[i] != '\0'; i++) {
        int j = 0;
        while (s2[j] != '\0' && s1[i + j] == s2[j]) j++;
        if (s2[j] == '\0') return (char *)(s1 + i);
    }
    return NULL;
}

int main(void) {
    const char *s = "the quick brown fox";
    char *p = my_strstr(s, "brown");
    if (p) printf("발견 위치: %ld, 이후: %s\n", p - s, p);
    else   printf("없음\n");
    return 0;
}
```

**시험 포인트:** 이중 루프. 바깥 루프는 시작 위치, 안쪽 루프는 일치 검사. `s2`가 끝까지 매칭되면(`s2[j] == '\0'`) 성공.

</details>

### D-2 ⭐⭐⭐ — 문자열 배열 정렬
영단어 5개를 입력받아 사전순으로 정렬해 출력하시오. (버블 정렬 + `strcmp` + `strcpy`)

<details>
<summary>👉 정답 예시</summary>

```c
#include <stdio.h>
#include <string.h>
#define N 5

int main(void) {
    char words[N][30];
    for (int i = 0; i < N; i++) {
        printf("단어 %d: ", i + 1);
        scanf("%29s", words[i]);
    }

    char tmp[30];
    for (int i = 0; i < N - 1; i++) {
        for (int j = 0; j < N - 1 - i; j++) {
            if (strcmp(words[j], words[j + 1]) > 0) {
                strcpy(tmp, words[j]);
                strcpy(words[j], words[j + 1]);
                strcpy(words[j + 1], tmp);
            }
        }
    }

    printf("\n정렬 결과:\n");
    for (int i = 0; i < N; i++) printf("%s\n", words[i]);
    return 0;
}
```

**시험 포인트:**
- 문자열끼리 직접 대입 불가 → `strcpy` 필요.
- 비교는 `strcmp` 결과의 부호로 판단.
- 만약 `char *words[N]`(포인터 배열) 이라면 **포인터만 교환**해도 되어 더 효율적.

</details>

```c
        {"문자열 끝을 나타내는 문자는? (\\0 형태로 입력)", "\\0"},
        {"문자열 길이 함수는?", "strlen"},
        {"문자열 비교 함수는?", "strcmp"}
    };

    int score = 0;
    char input[80];
    for (int i = 0; i < N; i++) {
        printf("\nQ%d. %s\n> ", i + 1, quiz[i][0]);
        gets_s(input, sizeof(input));
        to_lower(input);
        to_lower(quiz[i][1]);
        if (strcmp(input, quiz[i][1]) == 0) score++;
    }
    printf("\n점수: %d / %d (%d점)\n", score, N, score * 20);
    return 0;
}
```
