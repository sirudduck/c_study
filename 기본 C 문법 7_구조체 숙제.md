# 숙제: 구조체 (Struct)

---

## 📦 문제 1. 구조체 선언 및 출력

아래 조건에 맞게 프로그램을 작성하시오.

`Student`라는 구조체를 선언하라. 멤버는 다음과 같다.

- `char name[50]` : 이름
- `int id` : 학번
- `float gpa` : 학점 (0.0 ~ 4.5)

`main()` 함수에서 `Student` 변수를 하나 선언하고, 값을 직접 대입한 뒤 콘솔에 출력하라.

**출력 예시:**
```
이름: 홍길동
학번: 20241001
학점: 3.80
```

## 📦 문제 2. 구조체 배열 + 반복 입력

`Student` 구조체 배열을 크기 3으로 선언하고, 3명의 정보를 키보드로 입력받아 콘솔에 출력하라.

- `scanf()`를 사용하여 입력받는다.
- 이름, 학번, 학점 순서로 입력받는다.
- `for` 반복문을 사용하여 입력과 출력을 처리하라.

**출력 예시:**
```
--- 학생 목록 ---
[1] 홍길동 | 20241001 | 3.80
[2] 김철수 | 20241002 | 2.50
[3] 이영희 | 20241003 | 4.10
```

## 📦 문제 3. 구조체 배열을 인자로 받는 함수

`Student` 구조체 배열과 배열 크기를 인자로 받아 전체 학생 정보를 출력하는 함수 `printStudents()`를 작성하라.

- `main()`에서 3명의 학생 정보를 키보드로 입력받아 `printStudents()`를 호출하여 출력하라.
- 학점이 **2.0 미만**인 학생은 이름 옆에 `[경고]`를 표시하라.
- 함수 원형은 다음과 같다.

```c
void printStudents(Student* arr, int size);
```

**출력 예시:**
```
--- 학생 목록 ---
[1] 홍길동 | 20241001 | 3.80
[2] 김철수 | 20241002 | 1.30 [경고]
[3] 이영희 | 20241003 | 4.10
```

## 📦 문제 4. 구조체 포인터 + 검색 후 값 수정

`Student` 구조체 배열에서 학번으로 학생을 검색하여 해당 학생의 학점을 수정하는 함수 `updateGPAById()`를 작성하라.

- `main()`에서 3명의 학생 정보를 키보드로 입력받는다.
- 이후 수정할 학번과 새 학점을 입력받아 `updateGPAById()`를 호출하라.
- 해당 학번이 없으면 `"해당 학번의 학생을 찾을 수 없습니다."`를 출력하라.
- 함수 원형은 다음과 같다.

```c
void updateGPAById(Student* arr, int size, int targetId, float newGpa);
```

**출력 예시 (학번이 있는 경우):**
```
학번 20241002 학생의 학점을 2.50 -> 3.70 으로 수정했습니다.
```

**출력 예시 (학번이 없는 경우):**
```
해당 학번의 학생을 찾을 수 없습니다.
```

## 📦 문제 5. 구조체 배열 + 정렬 + 평균 계산 함수 복합 응용

`Student` 구조체 배열(크기 5)을 사용하여 아래 두 가지 함수를 작성하고, `main()`에서 순서대로 호출하라.

**작성할 함수:**

```c
// 학점 기준 내림차순으로 배열을 정렬하는 함수 (버블 정렬 사용)
void sortByGPA(Student* arr, int size);

// 전체 학생의 학점 평균을 반환하는 함수
float calcAverage(Student* arr, int size);
```

**조건:**
- `main()`에서 5명의 학생 정보를 키보드로 입력받는다.
- `sortByGPA()`로 정렬한 뒤 순위와 함께 콘솔에 출력한다.
- `calcAverage()`로 평균을 계산하여 출력한다.
- 학점이 **평균 이상**인 학생은 이름 옆에 `[우수]`를 표시한다.

**출력 예시:**
```
--- 학점 순위 ---
1위  최수아 | 20241005 | 4.30 [우수]
2위  이영희 | 20241003 | 4.10 [우수]
3위  홍길동 | 20241001 | 3.80 [우수]
4위  박민준 | 20241004 | 3.20
5위  김철수 | 20241002 | 2.50

전체 평균 학점: 3.58
```
