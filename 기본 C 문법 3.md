# Part 2: 배열 배우기

## 1. 배열이란?

**배열**: 같은 타입의 데이터를 **연속된 메모리 공간**에 저장하는 자료구조

**수학적 비유**: 수열 {a₁, a₂, a₃, ..., aₙ}
- a₁ → arr[0]
- a₂ → arr[1]
- aₙ → arr[n-1]

### 배열 vs 변수

```c
// 변수 5개
int score1, score2, score3, score4, score5;

// 배열 1개
int scores[5];  // 5개의 정수를 담을 수 있는 배열
```

**메모리 구조**
```
변수:
score1 [85]
score2 [92]
score3 [78]
score4 [95]
score5 [88]

배열:
scores[0] [85]
scores[1] [92]
scores[2] [78]
scores[3] [95]
scores[4] [88]
```

## 2. 배열 선언과 초기화

### 2.1 배열 선언

```c
자료형 배열이름[크기];
```

```c
int numbers[5];      // 정수 5개
double prices[10];   // 실수 10개
char grades[3];      // 문자 3개
```

**주의**: 
- 배열 크기는 **상수**여야 함
- 크기는 0보다 커야 함

```c
int n = 5;
int arr[n];     // ❌ C99 이전 버전에서는 오류 (변수로 크기 지정)

int arr[5];     // ✅ 올바름 (상수로 크기 지정)

#define SIZE 5
int arr[SIZE];  // ✅ 올바름 (매크로 상수)
```

### 2.2 배열 초기화

**방법 1: 선언과 동시에 초기화**
```c
int numbers[5] = {10, 20, 30, 40, 50};
```

**방법 2: 일부만 초기화 (나머지는 0)**
```c
int numbers[5] = {10, 20};  // {10, 20, 0, 0, 0}
```

**방법 3: 크기 생략 (자동 계산)**
```c
int numbers[] = {10, 20, 30, 40, 50};  // 크기 5로 자동 설정
```

**방법 4: 모두 0으로 초기화**
```c
int numbers[5] = {0};  // {0, 0, 0, 0, 0}
```

**방법 5: 선언 후 개별 초기화**
```c
int numbers[5];
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
numbers[3] = 40;
numbers[4] = 50;
```

## 3. 배열 요소 접근

### 인덱스 (Index)

배열의 각 요소는 **인덱스**(첨자)로 접근합니다.

**중요**: 인덱스는 **0부터 시작**!

```c
int scores[5] = {85, 92, 78, 95, 88};

scores[0]  // 85 (첫 번째 요소)
scores[1]  // 92 (두 번째 요소)
scores[2]  // 78 (세 번째 요소)
scores[3]  // 95 (네 번째 요소)
scores[4]  // 88 (다섯 번째 요소)
```

### 배열 요소 사용

```c
int scores[5] = {85, 92, 78, 95, 88};

// 읽기
printf("%d\n", scores[0]);  // 85

// 쓰기
scores[2] = 80;  // 78을 80으로 변경

// 연산
int sum = scores[0] + scores[1];  // 177
scores[3] = scores[3] + 5;        // 95를 100으로
```

### 주의: 인덱스 범위

```c
int arr[5];

arr[0]  // ✅ 유효
arr[4]  // ✅ 유효
arr[5]  // ❌ 범위 초과! (정의되지 않은 동작)
arr[-1] // ❌ 음수 인덱스! (정의되지 않은 동작)
```

## 4. 배열과 반복문

배열의 진정한 힘은 **반복문과 함께 사용할 때** 발휘됩니다!

### 예제 1: 배열 요소 입력

```c
#include <stdio.h>

int main(void) {
    int scores[5];
    int i;
    
    // 입력
    for (i = 0; i < 5; i++) {
        printf("%d번 학생 점수: ", i + 1);
        scanf("%d", &scores[i]);
    }
    
    // 출력
    printf("\n입력된 점수:\n");
    for (i = 0; i < 5; i++) {
        printf("%d번: %d점\n", i + 1, scores[i]);
    }
    
    return 0;
}
```

### 예제 2: 배열 합계와 평균

```c
#include <stdio.h>

int main(void) {
    int scores[5];
    int i, sum = 0;
    double average;
    
    // 입력
    for (i = 0; i < 5; i++) {
        printf("%d번 학생 점수: ", i + 1);
        scanf("%d", &scores[i]);
    }
    
    // 합계 계산
    for (i = 0; i < 5; i++) {
        sum += scores[i];
    }
    
    average = sum / 5.0;
    
    printf("합계: %d점\n", sum);
    printf("평균: %.2f점\n", average);
    
    return 0;
}
```

## 5. 5명 학생 성적 처리 (문제 1-1 해답)

```c
#include <stdio.h>

int main(void) {
    int scores[5];
    int i, sum = 0, max, min;
    double average;
    
    // 입력
    printf("=== 5명 학생 점수 입력 ===\n");
    for (i = 0; i < 5; i++) {
        printf("%d번 학생 점수: ", i + 1);
        scanf("%d", &scores[i]);
    }
    
    // 합계 계산
    for (i = 0; i < 5; i++) {
        sum += scores[i];
    }
    average = sum / 5.0;
    
    // 최대값, 최소값 찾기
    max = scores[0];
    min = scores[0];
    
    for (i = 1; i < 5; i++) {
        if (scores[i] > max) {
            max = scores[i];
        }
        if (scores[i] < min) {
            min = scores[i];
        }
    }
    
    // 결과 출력
    printf("\n=== 통계 결과 ===\n");
    printf("평균: %.2f점\n", average);
    printf("최고점: %d점\n", max);
    printf("최저점: %d점\n", min);
    
    return 0;
}
```

**실행 예시**

## 6. 숫자 뒤집기 개선 (문제 1-2 해답)

```c
#include <stdio.h>

int main(void) {
    int num;
    int digits[10];  // 최대 10자리
    int count = 0;   // 실제 자릿수
    int i;
    
    printf("숫자를 입력하세요: ");
    scanf("%d", &num);
    
    // 각 자릿수를 배열에 저장
    while (num > 0) {
        digits[count] = num % 10;
        count++;
        num = num / 10;
    }
    
    // 저장된 자릿수 출력 (이미 뒤집혀있음)
    printf("뒤집은 결과: ");
    for (i = 0; i < count; i++) {
        printf("%d ", digits[i]);
    }
    printf("\n");
    
    return 0;
}
```

**실행 예시**
```
숫자를 입력하세요: 12345
뒤집은 결과: 5 4 3 2 1
```

## 8. 버블 정렬 (Bubble Sort)

배열을 오름차순으로 정렬하는 알고리즘

**원리**: 인접한 두 요소를 비교하여 큰 값을 뒤로 보냄

```c
#include <stdio.h>

int main(void) {
    int arr[5] = {64, 34, 25, 12, 22};
    int i, j, temp;
    int n = 5;
    
    printf("정렬 전: ");
    for (i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    
    // 버블 정렬
    for (i = 0; i < n - 1; i++) {
        for (j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // 교환 (swap)
                temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
    
    printf("정렬 후: ");
    for (i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    
    return 0;
}
```

**실행 결과**
```
정렬 전: 64 34 25 12 22
정렬 후: 12 22 25 34 64
```

```c
#include <stdio.h>

int main(void) {
    double r1, r2, r3;
    double w1, h1, w2, h2;
    double area;
    const double PI = 3.141592;
    
    // 원 1
    printf("원1의 반지름: ");
    scanf("%lf", &r1);

    printf("원3의 반지름: ");
    scanf("%lf", &r3);
    area = PI * r3 * r3;
    printf("원3의 넓이: %.2f\n", area);
    
    // 직사각형 1
    printf("직사각형1 (가로 세로): ");
```

# Part 4: 함수 배우기

## 1. 함수란?

**함수**: 특정 작업을 수행하는 코드 블록에 **이름**을 붙인 것

**수학적 비유**: f(x) = 2x + 1
- **입력** (x): 매개변수 (parameter)
- **출력** (f(x)): 반환값 (return value)
- **계산** (2x + 1): 함수 본체 (body)

### C 언어의 함수

```c

    return 2 * x + 1;
}

// 사용

```

```c
    }
```

```c
    int max = arr[0];
    int i;
    
    for (i = 1; i < size; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }

}
```

## 2. 종합 예제: 성적 관리 시스템

```c
#include <stdio.h>

#define MAX_STUDENTS 10

int scores[MAX_STUDENTS];
int studentCount;

// 메뉴 출력
void printMenu(void) {
    printf("\n=== 성적 관리 시스템 ===\n");
    printf("1. 성적 입력\n");
    printf("2. 성적 출력\n");
    printf("3. 통계 보기\n");
    printf("4. 종료\n");
    printf("선택: ");

void inputScores(void) {
    int i;
    
    printf("학생 수를 입력하세요 (최대 %d명): ", MAX_STUDENTS);
    scanf("%d", &studentCount);
    
    if (studentCount > MAX_STUDENTS) {
        studentCount = MAX_STUDENTS;
    }
    
    for (i = 0; i < studentCount; i++) {
        printf("%d번 학생 점수: ", i + 1);
        scanf("%d", &scores[i]);
    }
    
    printf("입력 완료!\n");
}

// 성적 출력

    int i;
    
    if (studentCount == 0) {
        printf("입력된 성적이 없습니다.\n");
        return;
    }
    
    printf("\n=== 전체 성적 ===\n");
    for (i = 0; i < studentCount; i++) {
        printf("%d번: %d점\n", i + 1, scores[i]);
    }
}

// 통계 계산
void showStatistics(void) {
    int i, sum = 0, max, min;
    double average;
    
    if (studentCount == 0) {
        printf("입력된 성적이 없습니다.\n");
        return;
    }
    
    // 합계
    for (i = 0; i < studentCount; i++) {
        sum += scores[i];
    }
    average = sum / (double)studentCount;
    
    // 최대, 최소
    max = scores[0];
    min = scores[0];
    
    for (i = 1; i < studentCount; i++) {
        if (scores[i] > max) max = scores[i];
        if (scores[i] < min) min = scores[i];
    }
    
    // 출력
    printf("\n=== 통계 ===\n");
    printf("학생 수: %d명\n", studentCount);
    printf("평균: %.2f점\n", average);
    printf("최고점: %d점\n", max);
    printf("최저점: %d점\n", min);
}

int main(void) {
    int choice;
    studentCount = 0;
    
    while (1) {
        printMenu();
        scanf("%d", &choice);
        
        switch (choice) {
            case 1:
                inputScores();
                break;
            case 2:
                printScores();
                break;
            case 3:
                showStatistics();
                break;
            case 4:
                printf("프로그램을 종료합니다.\n");
                return 0;
            default:
                printf("잘못된 선택입니다.\n");

    }

}
```

```c
// 버블 정렬 함수
void bubbleSort(int arr[], int size) {
    int i, j, temp;
    
    for (i = 0; i < size - 1; i++) {
        for (j = 0; j < size - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // 교환
                temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

    printArray(numbers, 5);
```

### 함수 핵심 개념

| 개념 | 설명 | 예시 |
|------|------|------|
| 정의 | 함수 만들기 | `int add(int a, int b) { }` |
| 선언 | 함수 예고 | `int add(int a, int b);` |
| 호출 | 함수 사용 | `result = add(3, 5);` |
| 반환 | 결과 돌려주기 | `return sum;` |

### 배열 + 함수

```c
// 배열을 매개변수로 받기
```

## 🎓 요약

### 오늘 배운 것

**배열 (Array)**
- 같은 타입 데이터를 연속으로 저장
- 인덱스로 접근 (0부터 시작)
- 반복문과 최고의 조합
- 용도: 성적, 점수, 목록 등

**함수 (Function)**
- 코드 재사용
- 프로그램 구조화
- 매개변수와 반환값
- 용도: 계산, 판별, 출력 등

### 실전 패턴

```c
// 패턴 1: 배열 순회
for (i = 0; i < size; i++) {
    // arr[i] 처리
}

// 패턴 2: 계산 함수
int calculate(int a, int b) {
    return a + b;
}

// 패턴 3: 판별 함수
int isValid(int n) {
    return n > 0 && n < 100;
}

// 패턴 4: 배열 함수
void processArray(int arr[], int size) {
    for (i = 0; i < size; i++) {
        // 처리
    }
}
```
