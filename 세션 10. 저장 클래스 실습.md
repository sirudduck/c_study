# 실습 1 — 도서관 대출 카운터 (워밍업)

> **난이도**: ⭐☆☆☆☆ (쉬움)
> **소요 시간**: 약 30분
> **학습 목표**: `static` 지역 변수와 `const` 매개변수의 동작을 손으로 익힌다.

---

## 🎯 시나리오

여러분은 학교 **도서관의 도서 관리 시스템**을 만들고 있습니다. 이 시스템은 다음을 추적해야 합니다.

1. **이번 학기 누적 대출 건수** (한 번 올라가면 절대 안 줄어듦)
2. **현재 대출 중인 책의 권수** (반납하면 줄어듦)

이 정보들을 **함수 안에서만 관리**하고 싶습니다. 전역 변수로 빼면 다른 코드에서 실수로 건드릴 수 있으니까요.

**힌트**: 이게 바로 `static` 지역 변수가 빛나는 순간입니다!

---

## 📋 요구사항

다음 4개의 함수를 완성하세요.

1. `void borrow_book(const char *title);`
   - 책을 한 권 빌림
   - 누적 대출 건수 +1, 현재 대출 권수 +1
   - 책 제목을 출력 ("📖 [제목] 대출 완료")

2. `void return_book(const char *title);`
   - 책을 한 권 반납
   - 누적 대출 건수는 **변하지 않음**
   - 현재 대출 권수 -1
   - 책 제목 출력 ("✅ [제목] 반납 완료")

3. `int get_total_loans(void);`
   - 누적 대출 건수 반환

4. `int get_current_loans(void);`
   - 현재 대출 중인 권수 반환

## 🧩 코드 골격 — 빈칸을 채우세요

```c
#include <stdio.h>

/* ─────────────────────────────────────────────
 * 책을 빌리는 함수
 * ───────────────────────────────────────────── */
void borrow_book(/* 빈칸 1: 매개변수 선언
                    힌트: 문자열을 받지만 함수 내부에서 수정하지 않음 */)
{
    /* 빈칸 2: 누적 대출 건수를 저장할 변수
       힌트: 함수가 끝나도 값이 유지되어야 함, 0으로 초기화 */
    ______ int total_loans = 0;

    /* 빈칸 3: 현재 대출 권수를 저장할 변수
       힌트: 빈칸 2와 같은 키워드 */
    ______ int current_loans = 0;

    total_loans++;
    current_loans++;

    printf("📖 %s 대출 완료\n", title);
    printf("   (누적: %d건, 현재 대출 중: %d권)\n",
           total_loans, current_loans);
}

/* ─────────────────────────────────────────────
 * 책을 반납하는 함수
 * ───────────────────────────────────────────── */
void return_book(/* 빈칸 4: borrow_book과 같은 형식 */)
{
    /* 빈칸 5: 현재 대출 권수만 관리 (반납은 누적과 무관) */
    ______ int current_loans = 0;

    /* 빈칸 6: 권수 감소 처리 */
    ______________;

    printf("✅ %s 반납 완료\n", title);
    printf("   (현재 대출 중: %d권)\n", current_loans);
}

/* ─────────────────────────────────────────────
 * 메인 함수 (이미 작성됨, 수정하지 말 것)
 * ───────────────────────────────────────────── */
int main(void)
{
    borrow_book("C로 배우는 자료구조");
    borrow_book("운영체제론");
    borrow_book("디지털 논리 회로");

    return_book("C로 배우는 자료구조");

    borrow_book("선형대수학");

    return_book("운영체제론");

    return 0;
}
```
