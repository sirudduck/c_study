## 문제 5: 벡터 내적

3차원 벡터 두 개를 각각 배열로 저장하고, 내적(dot product)을 계산하세요.

**공식**: a·b = a₁b₁ + a₂b₂ + a₃b₃

**입력 예시**
```
벡터 a의 성분 (3개): 1 2 3
벡터 b의 성분 (3개): 4 5 6
```

**출력 예시**
```
벡터 a: (1, 2, 3)
벡터 b: (4, 5, 6)
내적 a·b = 1×4 + 2×5 + 3×6 = 32
```

**힌트**  
배열 2개, 대응하는 요소끼리 곱셈

## 문제 7: 분산 계산

정수 5개를 배열에 저장하고, 평균과 분산을 계산하세요.

**공식**: 
- 평균 μ = (x₁ + x₂ + ... + xₙ) / n
- 분산 σ² = [(x₁-μ)² + (x₂-μ)² + ... + (xₙ-μ)²] / n

**입력 예시**
```
정수 5개를 입력하세요: 10 12 14 16 18
```

**출력 예시**
```
데이터: 10 12 14 16 18
평균: 14.0
분산: 8.0
```

**계산 과정**
```
평균 = (10+12+14+16+18)/5 = 70/5 = 14
분산 = [(10-14)² + (12-14)² + (14-14)² + (16-14)² + (18-14)²] / 5
    = [16 + 4 + 0 + 4 + 16] / 5
    = 40 / 5
    = 8
```

**힌트**  
평균을 먼저 구한 후, 각 요소와 평균의 차이 제곱

## 문제 8: 행렬 덧셈

2×3 행렬 두 개를 1차원 배열로 저장하고 (row-major order), 두 행렬의 합을 계산하여 출력하세요.

**Row-major order 설명**
```
행렬 A = [ a₀ a₁ a₂ ]  →  배열 A[6] = {a₀, a₁, a₂, a₃, a₄, a₅}
         [ a₃ a₄ a₅ ]
```

**입력 예시**
```
행렬 A의 요소 6개를 입력하세요 (2×3):
1 2 3
4 5 6

행렬 B의 요소 6개를 입력하세요 (2×3):
7 8 9
10 11 12
```

**출력 예시**
```
행렬 A:
  1  2  3
  4  5  6

행렬 B:
  7  8  9
 10 11 12

행렬 A + B:
  8 10 12
 14 16 18
```

**힌트**  
배열 3개 필요 (A, B, C), 대응 인덱스끼리 덧셈
