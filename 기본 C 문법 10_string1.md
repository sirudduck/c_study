# Session 09 — 문자열 처리 Part 1: 문자열이란 무엇인가?

---

## 1. 문제 먼저 생각하기

지금까지 숫자를 다루는 방법은 배웠다.

```c
int score = 95;
float temperature = 36.5f;
```

그런데 이름, 주소, 메시지처럼 **글자로 이루어진 데이터**는 어떻게 저장할까?

```c
char name = ???   // "홍길동"을 어떻게 넣지?
```

`char`는 글자 **한 개**만 저장할 수 있다.

```c
char c = 'A';   // 한 글자는 가능
```

그렇다면 여러 글자를 저장하려면 어떻게 해야 할까?

---

## 2. 문자열 = 글자들의 줄서기

### 2.1 char 배열로 문자열 만들기

여러 글자를 저장하려면 `char`를 **배열**로 만들면 된다.

```
조리대(RAM) 위에 칸이 나뉜 접시를 올려놓는다고 생각해보자.

접시[0] | 접시[1] | 접시[2] | 접시[3] | 접시[4]
  'H'   |   'e'   |   'l'   |   'l'   |   'o'
```

```c
char name[6] = {'H', 'e', 'l', 'l', 'o', '\0'};
```

그런데 이렇게 한 글자씩 입력하는 건 불편하다.  
C는 더 편한 방법을 제공한다.

```c
char name[6] = "Hello";
```

두 코드는 **완전히 동일하다.**

### 2.2 널 문자(\0) — 접시 맨 끝의 "여기까지" 표시

위 배열을 자세히 보면 글자가 5개인데 배열 크기가 6이다.

```
name[0] | name[1] | name[2] | name[3] | name[4] | name[5]
  'H'   |   'e'   |   'l'   |   'l'   |   'o'   |  '\0'
```

마지막 칸에 `'\0'` 이라는 특수한 값이 들어간다.  
이것을 **널 문자(null character)** 라고 한다.

> **비유:**  
> 식당에서 여러 반찬을 일렬로 놓을 때, 맨 끝 반찬 옆에 **"여기까지입니다"** 라는 팻말을 세워둔다.  
> C의 문자열에서 `'\0'`이 바로 그 팻말이다.  
> 이 팻말이 없으면 어디서 반찬이 끝나는지 알 수 없다.

```
name[0] | name[1] | name[2] | name[3] | name[4] | name[5]
  'H'   |   'e'   |   'l'   |   'l'   |   'o'   |  '\0'  ← "여기까지"
```

**`'\0'`이 없으면 어떻게 될까?**

```c
char bad[5] = {'H', 'e', 'l', 'l', 'o'};   // '\0' 없음
printf("%s\n", bad);   // "Hello" 다음 메모리를 계속 읽어버림
```

```
결과 (예측 불가):
Hello▒▒▒▒▒▒▒▒▒ (쓰레기 값이 붙어서 출력됨)
```

`printf`의 `%s`는 `'\0'`을 만날 때까지 출력을 멈추지 않는다.  
`'\0'`이 없으면 **엉뚱한 메모리까지 출력**하게 된다.

### 2.3 배열 크기는 항상 글자 수 + 1

```c
char name[6] = "Hello";    // 5글자 + '\0' 1개 = 6칸 필요
char city[7] = "Seoul";    // 5글자 + '\0' 1개 = 6칸... 이지만 7로 여유 있게
```

> **규칙:** `char` 배열 크기는 항상 **저장할 최대 글자 수 + 1** 이상이어야 한다.

### 2.4 문자열 초기화 방법 비교

```c
// 방법 1: 문자열 리터럴로 초기화 (가장 일반적)
char name[20] = "홍길동";

// 방법 2: 빈 배열 선언 후 나중에 채우기
char name[20];
// 이 상태에서 name을 바로 쓰면 안 된다! (쓰레기 값)

// 방법 3: 빈 문자열로 초기화
char name[20] = "";    // name[0] = '\0', 나머지는 0
```

## 3. 문자열과 포인터

### 3.1 배열 이름은 첫 번째 칸의 주소

배열을 배울 때 배웠듯이, 배열 이름은 첫 번째 원소의 주소다.

```c
char name[6] = "Hello";

// name  ==  &name[0]  ==  'H'가 있는 주소
```

```
주소:   100   101   102   103   104   105
값:     'H'   'e'   'l'   'l'   'o'  '\0'
         ↑
        name (첫 번째 칸의 주소)
```

그래서 `printf("%s", name)` 은 사실 이렇게 동작한다.

```
1. name이 가리키는 주소(100)로 간다
2. 'H'를 출력한다
3. 주소를 1 증가(101)시킨다
4. 'e'를 출력한다
5. ... 반복 ...
6. '\0'을 만나면 멈춘다
```

### 3.2 함수에 문자열 전달하기

문자열(char 배열)을 함수에 전달할 때는 `char *` 또는 `char []`로 받는다.

```c
void print_name(char *name) {       // char name[] 과 동일
    printf("이름: %s\n", name);
}

int main(void) {
    char student[20] = "홍길동";
    print_name(student);            // 배열의 첫 주소를 전달
    return 0;
}
```

배열 전체를 복사하는 게 아니라 **첫 번째 칸의 주소만 전달**하는 것이다.  
함수 안에서 포인터를 통해 배열의 내용에 접근한다.

## 4. 문자열 기본 함수 — `<string.h>`

C는 문자열을 다루는 표준 함수들을 `<string.h>` 헤더에 모아두었다.  
사용하려면 반드시 포함해야 한다.

```c
#include <string.h>
```

---

### 4.1 strlen — 문자열 길이 재기

```c
size_t strlen(const char *s);
```

`'\0'` 이전까지의 글자 수를 반환한다.  
`'\0'` 자체는 길이에 포함되지 않는다.

```c
#include <stdio.h>
#include <string.h>

int main(void) {
    char name[20] = "Hello";
    printf("길이: %zu\n", strlen(name));   // 5
    return 0;
}
```

> **비유:**  
> 반찬 접시에서 "여기까지" 팻말 앞에 반찬이 몇 개인지 세는 것이다.

```
'H' | 'e' | 'l' | 'l' | 'o' | '\0'
  1     2     3     4     5    팻말
                              → strlen = 5
```

> **주의:** 배열 크기(`sizeof`)와 문자열 길이(`strlen`)는 다르다.

```c
char name[20] = "Hello";
printf("%zu\n", sizeof(name));    // 20 (배열 전체 크기)
printf("%zu\n", strlen(name));    // 5  (실제 글자 수)
```

---

### 4.2 strcpy — 문자열 복사하기

```c
char *strcpy(char *dest, const char *src);
```

`src`의 내용을 `dest`로 복사한다. (`'\0'` 포함)

```c
char src[20]  = "Hello";
char dest[20];

strcpy(dest, src);
printf("%s\n", dest);   // Hello
```

> **비유:**  
> 접시(src)에 담긴 반찬을 다른 빈 접시(dest)에 그대로 옮겨 담는 것이다.

> **주의:** `dest` 배열이 `src`보다 작으면 **버퍼 오버플로우** 발생.  
> `dest`는 항상 `src` 이상의 크기여야 한다.

```c
// 위험한 예:
char small[3];
strcpy(small, "Hello");   // small에 5글자가 들어가려다 터짐
```

**문자열을 대입할 때 `=` 연산자를 쓰면 안 된다:**

```c
char name[20];
name = "홍길동";    // 컴파일 오류! 배열에 = 대입 불가
strcpy(name, "홍길동");   // 올바른 방법
```

---

### 4.3 strcat — 문자열 이어붙이기

```c
char *strcat(char *dest, const char *src);
```

`dest` 문자열 끝에 `src`를 이어붙인다.

```c
char greeting[30] = "Hello, ";
char name[20]     = "World";

strcat(greeting, name);
printf("%s\n", greeting);   // Hello, World
```

> **비유:**  
> 접시(dest) 끝의 "여기까지" 팻말을 치우고, 다른 접시(src)의 반찬을 뒤에 이어 담은 뒤 팻말을 다시 세운다.

```
전:   'H','e','l','l','o',',',' ','\0'
후:   'H','e','l','l','o',',',' ','W','o','r','l','d','\0'
```

> **주의:** `dest` 배열이 결합 결과를 담기에 충분히 커야 한다.

---

### 4.4 strcmp — 문자열 비교하기

```c
int strcmp(const char *s1, const char *s2);
```

두 문자열을 비교한다.

| 반환값 | 의미 |
|--------|------|
| `0` | 두 문자열이 완전히 같다 |
| `양수` | s1이 s2보다 사전순으로 뒤에 있다 |
| `음수` | s1이 s2보다 사전순으로 앞에 있다 |

```c
char a[20] = "apple";
char b[20] = "apple";
char c[20] = "banana";

printf("%d\n", strcmp(a, b));   // 0  (같음)
printf("%d\n", strcmp(a, c));   // 음수 (apple이 banana보다 앞)
printf("%d\n", strcmp(c, a));   // 양수 (banana가 apple보다 뒤)
```

> **왜 `==`로 비교하면 안 되는가?**

```c
char a[20] = "apple";
char b[20] = "apple";

if (a == b) { ... }     // 틀림! 주소를 비교하는 것이지 내용을 비교하는 게 아니다
if (strcmp(a, b) == 0) { ... }   // 올바른 방법
```

`a`와 `b`는 서로 다른 배열이므로 주소가 다르다.  
`==`는 내용이 아니라 **주소**를 비교하기 때문에 항상 `0`(거짓)이 나온다.

---

### 4.5 strchr — 문자 찾기

```c
char *strchr(const char *s, int c);
```

문자열 `s`에서 문자 `c`가 **처음 등장하는 위치의 포인터**를 반환한다.  
찾지 못하면 `NULL`을 반환한다.

```c
char str[20] = "Hello, World";
char *p = strchr(str, ',');

if (p != NULL) {
    printf("쉼표 위치: %ld번째\n", p - str);   // 5
    printf("쉼표 이후: %s\n", p + 1);           //  World
}
```

> **비유:**  
> 접시에 놓인 반찬들을 앞에서부터 훑다가, 원하는 반찬을 발견하면  
> 그 **접시 칸의 위치(주소)** 를 알려주는 것이다.

---

### 4.6 strstr — 부분 문자열 찾기

```c
char *strstr(const char *haystack, const char *needle);
```

`haystack`에서 `needle`이 **처음 등장하는 위치의 포인터**를 반환한다.  
찾지 못하면 `NULL`을 반환한다.

```c
char str[50] = "오늘 날씨는 맑음입니다.";
char *p = strstr(str, "맑음");

if (p != NULL) {
    printf("찾았다: %s\n", p);   // 맑음입니다.
}
```

## 5. 핵심 정리 (Part 1)

```
문자열의 핵심 규칙
─────────────────────────────────────────
1. 문자열은 char 배열이다
2. 반드시 마지막에 '\0' (널 문자) 이 있어야 한다
3. 배열 크기는 최대 글자 수 + 1 이상
4. 대입은 strcpy, 비교는 strcmp (= 과 == 사용 금지)
5. 문자열 함수는 #include <string.h> 필요
─────────────────────────────────────────

주요 함수 요약
─────────────────────────────────────────
strlen(s)        글자 수 반환 ('\0' 미포함)
strcpy(dst, src) src → dst 복사
strcat(dst, src) dst 뒤에 src 이어붙이기
strcmp(s1, s2)   같으면 0, 다르면 비 0
strchr(s, c)     문자 c의 첫 위치 반환
strstr(s, needle) 부분 문자열의 첫 위치 반환
─────────────────────────────────────────
```
