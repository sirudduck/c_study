## 2. 구조체(struct)란?

**서로 다른 타입의 데이터를 하나로 묶은 사용자 정의 타입**이다.

```
struct Student
┌──────────────────────────────┐
│  char  name[20]   (20 bytes) │
│  int   id         ( 4 bytes) │
│  float score      ( 4 bytes) │
└──────────────────────────────┘
```

---

## 3. 구조체 정의와 변수 선언

### 3.1 기본 문법

```c
// 구조체 타입 정의 (설계도 작성)
struct Student {
    char  name[20];
    int   id;
    float score;
};

// 구조체 변수 선언 (실제 메모리 할당)
struct Student s1;
struct Student s2;
```

> **주의:** 구조체 정의는 **설계도**일 뿐이다. 실제 메모리는 변수를 선언할 때 할당된다.

### 3.2 typedef로 간편하게 사용하기

매번 `struct Student`라고 쓰는 것은 불편하다. `typedef`를 사용하면 타입 이름을 짧게 만들 수 있다.

```c
typedef struct {
    char  name[20];
    int   id;
    float score;
} Student;   // 이제 'Student'만으로 사용 가능

Student s1;  // struct Student s1; 과 동일
Student s2;
```

### 3.3 선언과 동시에 초기화

```c
Student s1 = {"홍길동", 20240001, 88.5f};
Student s2 = {.name = "이순신", .id = 20240002, .score = 95.0f};  // 지정 초기화
```

## 4. 멤버 접근: `.` 연산자

구조체의 각 데이터에 접근할 때는 **점(`.`) 연산자**를 사용한다.

```c
#include <stdio.h>
#include <string.h>

typedef struct {
    char  name[20];
    int   id;
    float score;
} Student;

int main(void) {
    Student s1;

    // 멤버에 값 대입
    strcpy(s1.name, "홍길동");   // 문자열은 strcpy 사용 (= 직접 대입 불가)
    s1.id    = 20240001;
    s1.score = 88.5f;

    // 멤버 값 출력
    printf("이름: %s\n",  s1.name);
    printf("학번: %d\n",  s1.id);
    printf("점수: %.1f\n", s1.score);

    return 0;
}
```

**실행 결과:**
```
이름: 홍길동
학번: 20240001
점수: 88.5
```

> **왜 `s1.name = "홍길동"`은 안 될까?**  
> `name`은 배열이다. 배열은 포인터처럼 동작하기 때문에 `=`로 직접 문자열을 복사할 수 없다.  
> 반드시 `strcpy()`를 사용해야 한다. (`#include <string.h>` 필요)

## 5. 구조체의 메모리 구조

```
Student s1 의 메모리 (총 28 bytes*)

주소(예시)  내용
─────────────────────────────────────────────────
0x100       name[0] ~ name[19]   (20 bytes) "홍길동\0..."
0x114       id                   ( 4 bytes) 20240001
0x118       score                ( 4 bytes) 88.5
─────────────────────────────────────────────────
```

> \* 컴파일러에 따라 **정렬(padding)** 때문에 실제 크기가 달라질 수 있다.  
> `sizeof(Student)`로 확인하는 것이 정확하다.

## 6. 구조체와 함수

### 6.1 값으로 전달 (Call by Value)

```c
void print_student(Student s) {   // 구조체 전체가 복사됨
    printf("[%d] %s - %.1f점\n", s.id, s.name, s.score);
}

int main(void) {
    Student s1 = {"홍길동", 20240001, 88.5f};
    print_student(s1);   // s1의 복사본이 전달됨
    return 0;
}
```

**문제점:** 구조체가 크면 복사 비용이 크다.  
멤버 수십 개짜리 구조체를 매번 복사하면 메모리와 시간이 낭비된다.

### 6.2 포인터로 전달 (권장)

```c
void print_student(Student *p) {   // 주소만 전달 (8 bytes)
    printf("[%d] %s - %.1f점\n", p->id, p->name, p->score);
}

int main(void) {
    Student s1 = {"홍길동", 20240001, 88.5f};
    print_student(&s1);   // 주소 전달
    return 0;
}
```

> **`->` 연산자:** 구조체 **포인터**로 멤버에 접근할 때 사용한다.  
> `p->id`  는  `(*p).id`  와 동일하다.

| 상황 | 연산자 | 예시 |
|------|--------|------|
| 구조체 변수 직접 접근 | `.` | `s1.id` |
| 구조체 포인터로 접근 | `->` | `p->id` |

### 6.3 구조체 수정 함수

함수 안에서 구조체 값을 변경하려면 반드시 포인터를 전달해야 한다.

```c
void add_bonus(Student *p, float bonus) {
    p->score += bonus;   // 원본 수정
}

int main(void) {
    Student s1 = {"홍길동", 20240001, 88.5f};
    printf("수정 전: %.1f\n", s1.score);  // 88.5

    add_bonus(&s1, 5.0f);
    printf("수정 후: %.1f\n", s1.score);  // 93.5
    return 0;
}
```

## 7. 구조체 배열

```c
#include <stdio.h>
#include <string.h>

typedef struct {
    char  name[20];
    int   id;
    float score;
} Student;

int main(void) {
    Student class[3] = {
        {"홍길동", 20240001, 88.5f},
        {"이순신", 20240002, 95.0f},
        {"강감찬", 20240003, 72.0f}
    };

    // 배열 순회
    for (int i = 0; i < 3; i++) {
        printf("[%d] %s - %.1f점\n", class[i].id, class[i].name, class[i].score);
    }

    return 0;
}
```

**실행 결과:**
```
[20240001] 홍길동 - 88.5점
[20240002] 이순신 - 95.0점
[20240003] 강감찬 - 72.0점
```

## 8. 구조체 배열과 포인터

```c
void print_all(Student *arr, int n) {
    for (int i = 0; i < n; i++) {
        printf("[%d] %s - %.1f점\n", arr[i].id, arr[i].name, arr[i].score);
        // 또는: (arr + i)->id, (arr + i)->name ...
    }
}

int main(void) {
    Student class[3] = {
        {"홍길동", 20240001, 88.5f},
        {"이순신", 20240002, 95.0f},
        {"강감찬", 20240003, 72.0f}
    };

    print_all(class, 3);   // 배열 이름 = 첫 번째 원소의 주소
    return 0;
}
```

## 9. 핵심 정리

| 개념 | 설명 |
|------|------|
| `struct` | 서로 다른 타입의 데이터를 하나로 묶는 사용자 정의 타입 |
| `typedef` | 타입에 별칭을 붙여 `struct` 키워드 없이 사용 가능 |
| `.` 연산자 | 구조체 **변수**의 멤버 접근 |
| `->` 연산자 | 구조체 **포인터**의 멤버 접근 (`(*p).x`와 동일) |
| `strcpy()` | 구조체 내 `char` 배열에 문자열 복사 시 사용 |
| 포인터 전달 | 크기가 큰 구조체를 함수에 전달할 때 권장 |
