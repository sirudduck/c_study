## 팁 — rand() / srand() 사용법

C에서 난수를 생성할 때 `stdlib.h`와 `time.h`를 사용한다.

```c
#include <stdlib.h>
#include <time.h>

srand((unsigned int)time(NULL));  // 프로그램 시작 시 1회만 호출
int n = rand() % 25 + 1;         // 1~25 사이 난수 생성
```

- `srand()`를 호출하지 않으면 매 실행마다 같은 숫자가 나온다.
- `rand() % N`은 0~N-1 범위이므로, 1~N이 필요하면 `+1`을 붙인다.
- `srand()`는 `main()` 시작 시 **반드시 1회만** 호출한다.

## Step 2 — 빙고판 랜덤 초기화 (20분)

### 과제

1~25를 **중복 없이** 랜덤 배치하여 빙고판을 초기화하라.

**요구사항**
- `initBoard()` 함수 작성
- `srand()`는 `main()` 시작 시 1회만 호출
- 중복 없이 1~25를 배치해야 함

**힌트 — 중복 없는 랜덤 배치 전략:**  
1~25를 배열에 순서대로 채운 뒤, 배열을 랜덤으로 섞는다 (Fisher-Yates shuffle).

```c
int nums[25];
for (int i = 0; i < 25; i++) nums[i] = i + 1;
for (int i = 24; i > 0; i--) {
    int j = rand() % (i + 1);
    int tmp = nums[i]; nums[i] = nums[j]; nums[j] = tmp;
}
```

### 체크포인트

- [ ] `srand()`를 `main()`에서 1회만 호출하는가
- [ ] 1~25가 중복 없이 배치되는가
- [ ] `initBoard()` 호출 후 `printBoard()`로 결과를 확인하는가
- [ ] 실행할 때마다 다른 배치가 나오는가
- [ ] 컴파일 및 실행 성공

### 과제

아래 함수 시그니처를 그대로 사용하여 구현하라.

```c

// 1~25를 랜덤 순서로 저장한 호출 목록 생성

```

**요구사항**
- `initCallList()` : Fisher-Yates shuffle로 1~25를 랜덤 순서로 섞어 저장
- `markNumber()` : 빙고판 전체를 탐색하여 `num`과 일치하는 칸을 자동 마킹
- `main()`에서 엔터를 누를 때마다 `callList`에서 번호를 순서대로 호출하고 마킹

**출력 형식:**
```
엔터를 눌러 번호를 뽑으세요...
호출된 번호: 17
```
