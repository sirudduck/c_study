# 실전 코딩 세션 — 영화 목록 관리 프로그램

> 단일 파일 / 2시간 / 사용 가능 개념: 변수·제어문·배열·함수·구조체·파일 I/O (텍스트)

---

## Step 1 — 구조체 정의 + 단일 데이터 출력 (20분)

### 과제

영화 1편의 정보를 구조체로 정의하고, 콘솔에 출력하라.

**요구사항**
- 필드: `title` (제목), `director` (감독), `year` (개봉연도), `rating` (평점)
- `main()`에서 구조체 변수 1개를 직접 초기화
- 출력 형식:

```
제목: Inception
감독: Christopher Nolan
개봉연도: 2010
평점: 8.8
```

### 체크포인트

- [ ] `typedef struct`로 정의했는가
- [ ] 문자열 필드를 `char[]`로 선언했는가
- [ ] `rating`을 `float` 또는 `double`로 선언했는가
- [ ] `%.1f`로 출력했는가
- [ ] 컴파일 및 실행 성공

---

## Step 2 — 구조체 배열 + 반복문으로 목록 출력 (20분)

### 과제

영화 여러 편을 배열로 관리하고, 반복문으로 전체 목록을 출력하라.

**요구사항**
- 구조체 배열로 영화 3편을 직접 초기화
- `for` 반복문으로 전체 목록 출력
- 출력 형식:

```
[1] Inception | Christopher Nolan | 2010 | 8.8
[2] Interstellar | Christopher Nolan | 2014 | 8.6
[3] The Matrix | Lana Wachowski | 1999 | 8.7
```

### 체크포인트

- [ ] 배열 크기를 상수(`#define MAX` 또는 `const int`)로 정의했는가
- [ ] 현재 저장된 영화 수를 별도 변수로 관리하는가
- [ ] 배열 크기가 아닌 현재 수만큼만 반복 출력하는가
- [ ] 컴파일 및 실행 성공

---

## Step 3 — 함수 분리 (추가 / 출력 / 검색) (25분)

### 과제

Step 2의 코드를 함수로 분리하고, 검색 기능을 추가하라.

**요구사항**
- `printAll()` : 전체 목록 출력
- `addMovie()` : 새 영화 추가 (사용자 입력, `scanf` 사용)
- `searchByTitle()` : 제목으로 검색 후 결과 출력

**검색 출력 형식:**
```
검색 결과: Inception | Christopher Nolan | 2010 | 8.8
```
또는
```
검색 결과 없음
```

### 체크포인트

- [ ] 함수 시그니처에 배열과 현재 개수를 인자로 넘기는가
- [ ] `addMovie()`에서 배열이 가득 찬 경우를 처리하는가
- [ ] 문자열 비교에 `==` 대신 `strcmp()`를 사용하는가
- [ ] `searchByTitle()`에서 검색 결과가 없을 때 메시지를 출력하는가
- [ ] 컴파일 및 실행 성공

## Step 4 — 텍스트 파일 저장 / 불러오기 (30분)

### 과제

영화 목록을 텍스트 파일로 저장하고, 프로그램 시작 시 불러오는 기능을 추가하라.

**요구사항**
- `saveToFile()` : `movies.txt`에 저장
- `loadFromFile()` : `movies.txt`에서 불러오기
- 파일 형식 (한 줄에 영화 1편):

```
Inception,Christopher Nolan,2010,8.8
Interstellar,Christopher Nolan,2014,8.6
```

### 체크포인트

- [ ] `fopen()` 반환값이 `NULL`인지 확인하는가
- [ ] 저장 시 `fprintf()`로 필드를 `,`로 구분하는가
- [ ] 불러오기 시 `fscanf()` 또는 `fgets()` + 파싱을 사용하는가
- [ ] `fclose()`를 반드시 호출하는가
- [ ] 파일이 없을 때 (첫 실행 시) 프로그램이 비정상 종료되지 않는가
- [ ] 컴파일 및 실행 성공

## Step 5 — 메뉴 루프로 전체 통합 (25분)

### 과제

지금까지 만든 함수들을 메뉴 루프로 연결하여 완성된 프로그램을 만들어라.

**요구사항**
- 프로그램 시작 시 파일에서 자동 불러오기
- 종료 시 파일에 자동 저장
- 메뉴 구성:

```
=== 영화 목록 관리 ===
1. 전체 목록 보기
2. 영화 추가
3. 제목으로 검색
4. 저장 후 종료
선택: 
```

### 팁

메뉴 루프의 기본 구조는 다음과 같다.

```c
int choice;
while (1) {
    printf("1. ...\n2. ...\n선택: ");
    scanf("%d", &choice);

    if (choice == 1) { ... }
    else if (choice == 2) { ... }
    else if (choice == 4) {
        // 저장 후 종료
        break;
    }
    else {
        printf("잘못된 입력입니다.\n");
    }
}
```

### 체크포인트

- [ ] `while(1)` 또는 `do-while` 루프로 메뉴를 반복하는가
- [ ] 유효하지 않은 메뉴 번호 입력 시 처리하는가
- [ ] 4번 선택 시 저장 후 루프를 종료하는가
- [ ] 프로그램을 재실행해도 이전 데이터가 유지되는가
- [ ] 전체 흐름 (시작 → 불러오기 → 메뉴 → 저장 → 종료) 이 오류 없이 동작하는가
- [ ] 컴파일 및 실행 성공
