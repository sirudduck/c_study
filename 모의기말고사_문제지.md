# 프로그래밍기초와실습 기말고사 모의시험 (문제지)

> **시험 범위:** 제10장 배열 · 제11장 포인터 (학교 강의 09~10주차)
> **시험 시간:** 80분 권장 (실제 시험 시간 기준)
> **총 문항:** 30문항 / 100점 만점
> **난이도 표기:** ⭐ 기초 / ⭐⭐ 표준 / ⭐⭐⭐ 응용 (총 3문항)
> **유의 사항:**
> - 모든 코드는 표준 C로 작성한다고 가정한다.
> - 시스템은 **64-bit Linux/gcc** 기준 (`int = 4B`, `double = 8B`, `pointer = 8B`).
> - 정답은 **모의기말고사_해설지.md**에 정리되어 있다. 먼저 모두 풀고 채점할 것.

**문제 8.** ⭐⭐ 다음 두 표현식의 의미 차이로 옳은 것은?
```c
*p++       (1)
(*p)++     (2)
```

- ① 둘 다 p가 가리키는 값을 1 증가시킨다.
- ② (1)은 p를 다음 칸으로 이동시키고, (2)는 p가 가리키는 값을 1 증가시킨다.
- ③ 둘 다 p를 다음 칸으로 이동시킨다.
- ④ (1)은 컴파일 오류이다.

**문제 19.** ⭐⭐

**문제 24.** ⭐⭐⭐ (어려움) 다음 코드의 출력을 16진수 그대로 적으시오.
(시스템은 little-endian, `int`는 4바이트로 가정)

```c
#include <stdio.h>
int main(void) {
    int data = 0x12345678;
    char *pc = (char *)&data;
    for (int i = 0; i < 4; i++)
        printf("%02X ", (unsigned char)*(pc + i));
    return 0;
}
```

> *힌트: little-endian은 낮은 자리 바이트가 먼저 메모리에 저장된다.*

**문제 30.** ⭐⭐⭐ (7점, 어려움) — 두 정렬 배열의 교집합

다음 함수를 **포인터 산술만 사용**(인덱스 표기 금지)하여 작성하시오.

```c
// 정렬된 두 배열 a (길이 na), b (길이 nb)에서 공통 원소를 c에 저장.
// 반환값: 교집합 원소 개수
int intersection(const int *a, int na, const int *b, int nb, int *c);
```

main에서 다음 입력으로 호출하시오.
- `a = {1, 2, 4, 5, 8, 10, 13, 15}` (길이 8)
- `b = {2, 3, 4, 7, 10, 11, 13, 14}` (길이 8)

예상 결과: `c = {2, 4, 10, 13}`, 반환값 4.

> *힌트: 두 포인터를 동시에 전진시키며 비교하면 O(na + nb)에 가능.*
